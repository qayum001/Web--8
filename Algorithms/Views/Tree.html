Format csv-text

Format csv-file: 1)Index;Name;Var1;Value1;Operator;Var2;Value2;NodeIndexs Index - number from 1 to infinity Name - name node of tree Var - name variable (can be None) Value - can be string or integer (can be None) Operator - < or <= or > or >= or == NodeIndexs - Child Nodes format (2_3_11) (can be None) 2)Var1;Value1;Var2;Number2;VarValue2;Next... Var1 - number 1 or 2 (can be None) Value1 - value (can be None)
